Pour le suivi des enfants jusqu'à l'âge adulte, MedinTux vous propose de visualiser les courbes biométriques.

Deux courbes sont actuellement disponible :

- la courbe de 0 à 3 ans, comprenant
* Poids
* Taille
* BMI
* Périmètre crânien

- la courbe de 3 à 22 ans
* Poids
* Taille
* BMI
* Croissance annuelle

Pour chacune des courbes les références de normalité (plus ou moins 2 deltas) sont indiquées selon le sexe et les âges .


Pré-requis :

* Renseignez les variables de poids, taille et périmètre crânien grâce au masque de terrain fournit avec MedinTux


Installation / Configuration :

* Dans un champs d'insertion, ajouter le script suivant :
{{::Graph( )}}



Résultat en images (en 1 clique) :

Pour la courbe de 0 à 3 ans, l'axe de l'âge est en mois.



Pour la courbe de 3 à 22 ans, l'axe de l'âge est en années


Limitations :
Vous devez absolument utiliser le masque de terrain fournit avec MedinTux.
