1 Logiciel apitux :

1.1 Généralités

Cet assistant de la suite MedinTux, permet de récupérer et intégrer automatiquement des résultats biologiques provenant d'Apicrypt

  • doc à écrire et finir
    Ce logiciel est destiné à l'intégration de données Apicrypt décryptées.
    • Les fichiers doivent se trouver dans le dossier "apitemp".
    Pour cela il faut configurer la sortie des fichiers décryptés dans "Options>Outils>Préférences>Exportation" de la boîte mail Apicrypt.
  • Quelques informations utiles :
    • Pour l'installation de l'intégrateur de données apicrypt "Apitux", placez le dossier à côté du dossier DrTux.
    Placez le contenu (Apitux) du dossier Menu dans DrTux/bin/ressources/Menus afin qu'un bouton apparaisse dans le logiciel DrTux.
    Il faut programmer Apicrypt afin que les fichiers décryptés soient envoyés dans apitux/bin/apitemp sous format resutextxxx.txt. Voir l'aide d'Apicrypt.
    Lors de l'utilisation les fichiers imprimés sont sauvegardés dans apitempsauve. Il faut penser à les purger de temps à autre.

    Lors de l'intégration, si la fenêtre de correction s'ouvre, il faut en général corriger le nom ou le prénom ou bien créer un patient qui n'existerait pas encore dans MedinTux.
    L'intégration se fait selon les critère HPRIM soit 1 numéro de dossier, 2 nom, 3 prénom , etc...
    Les titres des fichiers intégrés peuvent se programmer à partir d'un élément affiché du fichier resutext d'Apicrypt. Par exemple si le nom du correspondant s'affiche en ligne 10, vous pouvez faire un alias avec ce correspndant (par exemple tous les cardios seront en titre CARDIO).
