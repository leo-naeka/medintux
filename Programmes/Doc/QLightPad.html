1) Introduction :

QLightPad est un petit éditeur de texte libre sans prétention permettant 
d'éditer des textes .html simples, d'importer des fichiers .rtf simples, 
et aussi de petits textes ASCII.
permet de placer des ancres , liens , images afin de créer 
de petits sites internet simples.
Ne sont pas supportés ni les 'frames' ni les tableaux et encore moins les
sophistications avancées des composeurs de pages Web.....

J'avais besoin d'avoir sous Linux un petit éditeur semblable à Wordpad
sous Windows, permettant de reprendre mes documents .rtf
Si cela peut vous servir à quelque chose, eh bien, servez vous....
QLightPad Fonctionne aussi sous environnement Windows où il
a été compilé avec succés.
Seule l'installation sous Linux sera décrite.

2) Préalable à l'intallation :

Il vous faut un environnement pouvant compiler des programmes QT :
 

  • QT version >= 3.1.2 installé en version développement donc
    • avec les répertoires includes de QT et les fichiers headers .h
    allant avec.
  • Les outils QT dont qmake, uic, moc etc ....
  • GCC 3.2 et tous les outils classiques de base comme make etc.

Bbref il vous faut une machine pouvant compiler des programmes QT.

3) Installation :

    a) Extraire l'archive .zip ou tar.gz 
  • Téléchargez et placez l'archive de QLightPad (.zip ou tar.gz) 
    • dans le répertoire de votre choix.  
  • Ouvrez une console dans le répertoire de votre choix
    • (pour les paresseux, CTRL T avec Konqueror sur le bon répertoire),
    tapez :
      tar -zxvf qlightpad.tar.gz  pour les fichiers avec l'extension .tar.gz
      unzip qlightpad.zip  pour les fichiers avec l'extension .zip 

      cela générera des répertoires avec la hiérarchie suivante :

      repertoireDeVotreChoix
                            /qlightpad_dev
                                          /MedinTuxTools
                                          /qlightpad/src
                                                    /bin
                                                        /QLightPadSys

    b) Compilation du programme avec le misérable setup

       Ouvrez une console (elle doit déjà être ouverte si vous avez suivi
       l'étape précédente).
       Placez-vous dans le répertoire: qlightpad_dev/

           cd qlightpad_dev 
       
           puis tapez :
                    ./setup

          et, normalement, cela devrait créer et lancer QLightPad 

    c) Compilation du programme pas à pas

       Ouvrez une console
       Placez-vous dans le répertoire : qlightpad_dev/qlightpad/src 
       tapez :

      qmake src.pro
      make
      make clean
      rm Makefile
      cd ../bin
      strip -s QLightPad

          si tout se passe bien, cela devrait 
       générer dans le répertoire :
       
            /qlightpad_dev/qlightpad/bin

          le fichier exécutable suivant : 
       
       QLightPad .         Et voilà; c'est lui qui est à exécuter !!!
       testez-le avec par exemple la commande suivante :

       ./QLightPad ../QLightPadSys/Welcome.html

       
       Dans le même répertoire que l'exécutable, 
       il doit y avoir le répertoire : /QLightPadSys 
       contenant le fichier :           QLightPad.png
          Je vous laisse le soin de recopier l'exécutable : QLightPad avec 
       son répertoire : QLightPadSys
       au lieu d'installation de votre choix, et d'effacer si vous le désirez
       le repertoire de création : qlightpad_dev
